# surface_laptop_3_sonoma_opencore

### 适用于 surface laptop 3 1035G7

### Macos 版本：macOS Sonoma 14beta 7（23A5337a）
### opencore 版本：0.9.4

## 注意事项：    
- Lilu.kext 须使用 v1.6.6 版本 （macOS Sonoma 14beta 7（23A5337a）），原因未知


## 能用：
- cpu
- igpu 显存1536MB，解码正常
- 触控板
- 扬声器
- 电池
- 触摸屏
- 蓝牙
- 内置摄像头
- 3.5mm耳机
- 亮度调节
- wifi
- 键盘背光
- CPU变频
- 睡眠唤醒
- hidpi ( 1368 x 912 )



## 不完美：
- 睡眠后唤醒很慢
- cpu温度较高
- usb 未定制


## 待测试
- usb 3.0 
- usb type-c
- 随航
