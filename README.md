# surface_laptop_3_sonoma_opencore0.9.4

### 适用于 surface laptop 3 1035G7

### Macos 版本：macOS Sonoma 14beta 7（23A5337a）

## 能用：
- cpu
- igpu 显存1536MB
- 触控板
- 扬声器
- 电池

## 不能用：
- 亮度调节
- wifi
- 蓝牙
- 睡眠唤醒

## 待测试

- 键盘背光
